Mest数据仓库工程师

Posted on February 19, 2023

薪资范围：[20k-40k]
工作地点：远程
年限要求：5-10 年

公司介绍

Mest 推崇异步交流与有效沟通。我们的团队成员通常是远程工作 (北京时间)，并没有严格的签到、打卡要求，没有加班文化，以工程产品输出为考核目标，每周会议在 2 次左右 (工程团队)。

我们在工程工具上的选型注重与社区接轨，并有一定的技术品味：

技术领域内的 Review 集中在 GitHub Pull Request
内部沟通使用 Slack / Google Meet
项目管理依赖 Linear ，文档沉淀于 Notion
云服务设施包含 AWS / Cloudflare / Heroku / Fly
我们推崇使用正版软件，可以满足个人的工具设施要求 (费用)

职位描述

  • 结合业务设计、规划与实施区块链相关数据仓库与服务设施
  • 分析并构建合理的数据模型与 ETL 流程等，持续建设与改进高性能数据服务
  • 参与业务的数据调研、分析，满足上游数据需求，支撑部分业务决策
  • 总结与维护技术文档，推动发展优质产品

职位要求

  • [重要] 优秀的异步协作、沟通能力，在远程工作中有较高的自我驱动力，能够主动推动任务进展
  • 丰富的大数据平台经验，对各类生态与解决方案有实施和考量经验
  • 了解区块链数据的业务模型，能够在指定场景下开发、优化 ETL
  • 至少熟练掌握 NodeJS / Java / Python 一种语言
  • 熟悉 Hadoop 生态并有相关实践优先
  • 在链上数据、区块链金融领域有经验者优先 (非硬性要求)

远程工作要求

Mest 注重人性化、友善地、包容并重地工作文化，我们相信每个工程师产生的价值来自于文化的认同感，自由、开放地环境是非常重要的一部分。

我们列出的职位均支持远程工作，但仍旧需要您具备一定的远程工作能力：

具备良好的自我驱动力、协作能力
能够合适的调节工作生活状态，保持合理的产出
有一定的异步交流、文档协作经验

联系方式

ashu@mest.io

公司介绍

BYDFi 是一家致力于为海外互联技术提供解决方案的高科技公司，主要业务为海外移动应用软件的开发等。我们拥抱互联网精神，以人才、技术、产品为中心，深信科技与创新会给客户带来更具价值的服务体验，专注为国际客户提供卓越的服务。
公司在新加坡、日本、台湾等多地设有分公司，团队汇集海内外不同领域的专业人才，涵盖互联网以及金融行业等；能够提供一定海外发展机会。

职位描述

1 、负责区块链交易所产品设计、实施，预见项目风险，把控项目范围，解决产品实现过程中的相关问题；
2 、负责协调整合资源，与协作部门保持良好沟通，确保产品顺利上线；
3 、收集市场反馈，跟踪产品运营数据，不断完善和优化产品，提升产品的用户体验；
4 、负责产品上线后产品迭代，包括但不限于业务逻辑优化，用户体验改善、交互优化、新功能规划和设计；

职位要求

1 、有音乐产品或者股票交易（雪球、东方财富）经验优先；
2 、责任心强，热爱金融科技行业；
3 、基础的英语阅读和优秀的学习借鉴能力、逻辑思维能力；
4 、有较强的产品 owner 意识及自驱能力，愿意接受挑战，能在较大压力下保持工作激情。
