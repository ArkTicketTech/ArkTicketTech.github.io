秘猿科技rust工程师

Posted on February 20, 2023

薪资范围：[30k-50k]
薪资月份：16薪
工作地点：远程
年限要求：5-10 年

公司介绍

杭州秘猿科技有限公司（ Cryptape Co.,Ltd.）的使命是用技术创造信任，为未来数字世界构建新的基础设施和服务。秘猿科技成立于 2016 年 ，2017 年获得招银国际战略投资，是一家以纯研究与工程为导向的技术公司，在区块链领域实力深厚。

我们具有丰富的区块链生态实践经验。从 2011 年开始参与或主导各种区块链项目，包括世界第一个开源的数字资产交易平台貔貅 Peatio ；星火矿池 Spark pool ； 以太坊 Ethereum ；以太坊爱好者社区 Ethfans ； 开源区块链底层内核 CITA ；与招商银行等主流金融机构进行的金融+区块链合作；以及现在正在开发的 Nervos CKB 、Godwoken 、Axon 等等，涵盖了区块链生态的各个方面。

我们的网站：https://www.cryptape.com/
我们的代码： https://github.com/cryptape
我们的博客：https://blog.cryptape.com/
我们的研究：https://blog.cryptape.com/series/research

职位描述

  • Design and implement the inter-blockchain communication solution with Cosmos IBC for Nervos Network.
  • Research different blockchains and explore the possibility of connecting them with Nervos Network.

职位要求

  • 3+ years experience in software development.
  • Proficiency in Rust/Golang language.
  • Familiar with distributed operating systems, experience with cloud native, cloud computing will be a plus.

岗位亮点

  • 远程办公:无论你在哪里,我们都可以并肩奋斗!
  • 弹性工作:无论你是早睡早起的好孩子还是不怕黑眼圈的夜猫子，我们都可以并肩奋斗！
  • 商业保险:除五险一金外,额外购买商业保险为你提供更多保障！
  • 成长机会:我们关注你的个人成长，线上学习平台线下活动交流都在碗里！

Posted on February 20, 2023

薪资范围：[30k-50k]
工作地点：远程

公司介绍

Xmind Ltd.是思维导图和头脑风暴软件的开发商，聚焦于提供优秀的用户体验，目前拥有 iOS, Android, Mac, Windows, Linux 多端思维导图产品，各具特色，满足了不同用户的使用场景需求。我们以提供优秀的生产力工具和服务，帮助人们节省时间并提升效率为目标。
我们每天为数十个国家的团队服务，从初创团队到世界 500 强企业。旗下产品深受好评，常年排名 Mac/iOS App Store 效率榜单前列，多次获得 App Store 推荐。

职位描述

  • 开发与维护 Xmind 产品的功能；
  • Xmind 创新功能的开发与维护；
  • 运用客户端编程语言（ JavaScript ）解决产品需求；
  • 配合交互设计师和视觉设计师，打造优质的用户体验；
  • 配合后端工程师，完成服务代码的有效对接。

职位要求

  • 一年以年以上全栈开发经验，具备云存储及协同项目经验；
  • 精通 HTML/CSS/Javascript/Typescript/Node.js 的使用，熟悉现代原生 JS 的开发使用，精通 vue3 等 MVVM 框架的开发和使用
  • 具备对云服务（ AWS 、阿里云、腾讯云、Google Cloud 等）了解和使用经验，具备 AWS CloudFormation/ELB/DynamoDB 等服务的开发部署经验优先；
  • 具备一定的前端自动化开发部署经验、熟悉 Docker 相关技术栈的使用；
  • 具备良好的英语读写习惯和追求代码质量，熟练掌握常见的设计模式和数据结构算法
  • 能快速学习、有效沟通，有团队合作精神，对新技术或不熟悉的领域有探索精神；
  • 掌握操作系统和网络通信原理，理解 HTTP/HTTPS 协议、文件系统、并发、分布式，锁等概念；
  • 其他优先考虑范围：有其他跨平台应用开发经验；有浏览器性能调试及优化经验；有 SVG 或 Canvas 等图形化界面开发经验；对软件的安全性或可访问性有一定研究。

Hashdata CloudBerry内核团队-数据库内核工程师

Posted on February 19, 2023

薪资范围：[30k-80k]
薪资月份：12薪
工作地点：北京/上海/杭州/深圳/远程
年限要求：3-5 年

职位描述

与数据库技术大拿一起，共同构建下一代世界级云数据库内核引擎，包括云原生查询优化器 / 执行器 / 存储引擎，使其能够高效处理 PB 级乃至 ZB 级的数据量。同时，你将有机会参与开源数据库项目，参加各种技术会议和发表技术文章。

职位要求

  • 计算机或相关专业，本科及以上学历，热爱技术，扎实的编程功底和技术基础，熟悉 Linux 下的编程环境下的 C/C++编程。
  • 渴望快速成长，具备快速学习能力和优秀的问题分析能力，自我驱动力强，良好的团队协作与沟通。
  • 没有相关经验但很喜欢也可以试试。比如，笔者就是从前端工程师转行数据库内核的。最近做的工作已经有一定复杂度，可参考笔者最近的文章。

加分项

  • 深入了解或参与过数据库、分布式系统者优先，包括但不限于 Postgres, MySQL, TeraData, Greenplum, Impala, leveldb, rocksdb, Presto, Clickhouse 等。
  • 深度参与过开源项目或自己有代表开源作品者优先。
  • 参与过复杂系统开发者优先，包括但不限于 Linux 内核，网络协议栈等。
  • 学习过 MIT 6.824 ，CMU 15445 等课程者优先。

工作环境

  • 此岗位可远程协同办公，我们的员工目前分布在国内各个城市，期待可以早日开拓海外市场。
  • 每年都有 20 天年假，拒绝 996 。
  • 每周五有下午茶。
  • 内核团队每周都有内部技术分享；有的同事有 3k star 的个人开源项目，有的同事有 2w+ 技术公众号粉丝。当然，像我这样只会玩泥巴的同事也是有的，好在大佬们都很愿意和我一起结对编程～
  • 可以从一件小事了解我们组的氛围和 Leader 。
